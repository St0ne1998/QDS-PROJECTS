QDS PROJECTS WEBSITE
====================

Project Name:
QDS Projects – Construction & Renovation Website

Description:
This project is a premium, modern, and fully responsive website built for QDS Projects, a construction and renovation company based in Nelspruit, Mpumalanga, South Africa. The website showcases the company's services, portfolio, testimonials, and contact information in a visually appealing and professional layout.

--------------------------------------------------

FEATURES
--------

• Fully responsive design (desktop, tablet, mobile)
• Modern UI/UX with premium styling
• Interactive hero slideshow
• Custom cursor animation
• Smooth scrolling navigation
• Services section with hover effects
• Project portfolio gallery
• Testimonials section
• Step-by-step process section
• Contact form UI (frontend only)
• Embedded Google Map location
• Clean and structured code

--------------------------------------------------

TECHNOLOGIES USED
-----------------

• HTML5
• CSS3 (Advanced styling & animations)
• JavaScript (Vanilla JS)
• Google Fonts (Playfair Display & Outfit)
• Unsplash (for demo images)

--------------------------------------------------

PROJECT STRUCTURE
-----------------

qds.html   → Main website file (contains HTML, CSS, and JavaScript)

Note: This is a single-page website with all styling and scripting embedded in one file.

--------------------------------------------------

HOW TO RUN
----------

1. Download or clone the repository
2. Locate the file: qds.html
3. Open the file in any modern web browser (Chrome, Edge, Firefox)

No installation or dependencies required.
